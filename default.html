The Machine Intelligence Cup

The Machine Intelligence Cup is a competition that will be held in the summer of 2014 in Montreal, QC. It will consist of a game, called Hex, for which participants will write artificial players. The programs written by participants will then play each other, recieve rankings, and the top players will compete at an event, with the final games displayed on a screen in front of a live audience!

The back-end of the game is complete, and a preliminary GUI has been written. Here is a quick tutorial explaining the basics of how it works.

Below is a demo of the game, using the some copy/pasted output from the back end into the latest iteration of the front end. In this game, the players played according to the following logic:

  • If the enemy base is in sight, attack it
  • Otherwise, choose a move at random
